Comment jouer

Compte
  • Changez votre nom d'utilisateur
  • Apprenez à utiliser l'application
  • Supprimez votre compte (nom et médailles !)
 Catégories
  • Choisissez un jeu
  • Dans Lettres, utilisez les lettres pour trouver un mot ou pays
  • Dans Chiffres, utiliser les chiffres pour trouver un nombre
 Jeux
  • Motus
    • Trouvez un mot, indices : première lettre, couleurs rouge et jaune
  • Drapeau
    • Trouvez un pays, indices : drapeau, couleurs rouge et jaune
  • Meli Melo
    • Trouvez un mot, indices : lettres mélangées et couleur rouge
  • Juste Prix
    • Trouvez un nombre, indices : flèches vertes si + et rouge si -
Médailles
  • Gagnez des médailles en jouant
C'est parti
